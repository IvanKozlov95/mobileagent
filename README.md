# nodeBlogApi

Rest api for simple blog on NodeJs with express and mongodb.

To deploy run in console

```
git clone https://github.com/IvanKozlov95/mobileagent
npm i
node index.js
```

To create test db with admin user run
```
node createdb.js
```
